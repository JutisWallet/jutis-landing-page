Quick Start Guide

1. Install the SDK

Get started by installing Jutis SDK via npm or yarn.

npm install @jutis/sdk

2. Initialize Jutis

Import and initialize the SDK with your API key.

import { Jutis } from '@jutis/sdk';

const jutis = new Jutis({
  apiKey: 'your-api-key'
});

3. Connect Wallet

Request wallet connection from the user.

const wallet = await jutis.connect();
console.log(wallet.address);
