Installation

Browser Extension

Jutis is available as a browser extension for Chrome, Firefox, and Edge.

  1. Visit the Chrome Web Store or Firefox Add-ons
  2. Search for "Jutis Wallet"
  3. Click "Add to Browser" to install
  4. Click the Jutis icon in your toolbar to get started
