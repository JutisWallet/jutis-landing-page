## About

Jutis is a modern web extension wallet designed for the Canton network. It provides bank-grade security for managing digital assets with an intuitive, futuristic interface.

### Key Features

- **Non-Custodial Security** — Your private keys never leave your device
- **Multi-Chain Support** — Manage assets across Canton and connected blockchains
- **Instant Swaps** — Token swaps with competitive rates
- **Biometric Authentication** — Face ID, Touch ID, Fingerprint support
- **QR Payments** — Fast and easy payments
- **Encrypted Backup** — Secure seed phrase recovery
